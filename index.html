What is pubblz?

pubblz is an editor for writing and publishing on the web.

Text content lives in fragments of markdown. Designs live in npm packages, created by experts using HTML, CSS, javascript etc. Images live in dropbox or the like.

By cleanly separating the source from the design, pubblz insulates content owners from the complexities of modern web standards.

Preview a sample presentation or documentation, or visit github to check out the source for this page..

pub-server

pub-server is the technology underlying pubblz.

Components include

  • a static site generator
  • an editor
  • a web server

pub-server runs on node.js. The generator and editor run in the browser, enabling instant preview.

Markdown content is stored in text files - ready for github.

The architecture is database-free. All code is MIT licensed.

See doc for more details.

Designs and Extensions

pub-server leverages the npm javascript module ecosystem.

Developers and designers can draw from the awesome selection of npm modules to create new layouts, generator plugins, server plugins, integrations, and browser scripts.

They can then publish their packages via the npm registry, to distribute their designs and extensions to pubblz users.

Examples: this design, documentation design, slideshow design, icon package, font package, github integration, google authentication, authorize.net payments.

Static Hosting

Generating HTML at publishing time just makes sense!

Static websites are simpler, more secure, more reliable, more scalable and lower cost than database-driven CMS hosting.

Many static generated sites are now hosted on services like GitHub pages, Amazon S3, and netlify.

Static hosting can be combined with CDN services like Amazon CloudFront or CloudFlare.

Real-time interactions like authentication and payments can be routed to separate back-end services.
