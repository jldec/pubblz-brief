pubblz Business Model Canvas

pubblz Value Proposition

Enable people to communicate by publishing on the web

Using designs crafted in HTML by designers and developers and accessed by readers using a browser

Replacing documents shared via email

With responsive fresh experiences across modern devices

Customers

consumers

Publish photos, videos, live-streams, announcements, events, albums, shortform and longform posts, school projects, games ...

business people

Publish presentations, websites, reports, project pages, team pages, analysis.

designers

Can use pub-server as a tool to rapidly create HTML and CSS layouts, not necessarily just for other pub-server users.

developers

Publish documentation, design docs, ideas, gists. Later also as a development environment.

Partners

designers

Quality of the HTML output designs will be critical for the perceived value of the service. Pubblz will need to showcase and reward designers.

Once a critical mass of users is achieved, pubblz can offer a design marketplace for top-tier designers to sell their work and support their users.

developers

Open Source Software developers are essential for the evolution of the platform and its capabilities. Pubblz already uses some fantastic libraries internally. Real magic is possible when developers start writing plugins.

service-partners and suppliers

Pubblz has been designed to leverage infrastructure as a service from providers like Amazon and Cloudflare, and will integrate with best-of-breed 3rd party services like Stripe for payments.

Competitors

Squarespace, Wix, Weebly, Shopify, Jimdo

SaaS self-service website builders. Offer wysiwyg editing with professionally designed templates. Rapidly taking over lower-end web design and hosting business from agencies and even Wordpress. Profitable and spending heavily on marketing.

Limitations: Proprietary, integrated CMS, limited extensibility, no self hosting

Wordpress

Wordpress is the dominant open source tool for website creation. Originally designed for blogging. Significant ecosystem of themes and plugins and hosting providers.

Dated architecture based on PHP and mySQL. Vulnerable to security breaches because of integrated admin/CMS.

Google Docs

Google Docs was the first to offer real-time co-editing features. Originally modeled after Microsoft Office with similar content types, import/export features, and print capabilities - now also supports forms, add-ons. Google docs is popular with schools.

Others

Drupal, Medium, Ghost, slides.com, cloudcannon

Channel: pubblz.com

pubblz.com will be the primary revenue channel. Customers signup online and pay with credit card. There will be a free trial period and referral discounts.

The Basic plan (not free) includes hosting, custom domain name and private sharing of content.

The Professional plan includes https, multi-user editing, and forms.

A-la-carte services payment processing, and support.

free tier

Pubblz is free for developers to self-host e.g. on GitHub, Heroku or AWS.

A consumer-free offering would definitely help to promote the service and accelerate adoption, but also changes the "dynamics" of the business.

This may make sense in partnership with hosting providers, or in the context of an acquisition.

Editor app

App-store sales of an editor could provide an additional source of non-recurring revenues, or serve to attract more subscribers. Certain editor features also work better in an app.

  • requires command line, javascript, HTML, CSS and other technical skills
  • in production use at www.fmctraining.com

  • missing CSS and HTML editor, single-click installer
  • investigating plugin for Atom (GitHub editor)

consumer product

  • missing consumer-grade UX
  • Medium-style wysiwyg editor
  • dropbox integration
  • image asset pipeline - basic resize

awareness

  • missing early adopter marketing
  • developers and designers

operations

  • signup, identity service
  • automated setup with CDN
  • devops, monitoring
  • documentation & support
